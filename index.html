FFSampledSP

FFmpeg based service provider for javax.sound.sampled.

Download this project as a .zip file Download this project as a tar.gz file

Welcome!

FFSampledSP is a free implementation of the javax.sound.sampled service provider interfaces based on FFmpeg, a complete, cross-platform solution to record, convert and stream audio and video.

Its main purpose is to decode audio files or streams to linear PCM. Yes! This package lets you play all the OGGs and FLACs you want. In Java, on Windows or OS X.

FFSampledSP makes use of the tagtraum FFmpeg package and is licensed under LGPL 2.1.
